# cloudbyte
Ta-Feng grocery dataset model to forecast the sales of any item in the inventory

#-------------------------------------------
* The code should work with both Python 2.x and 3.x, I have tested it with 2.7.12
* Required Python packages
** Numpy
** Scipy
** Matplotlib
** Pandas
** Scikit
** Statsmodels

# Steps

* cd src
* python model.py
